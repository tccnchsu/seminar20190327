[ 運作原理 ]
1. Client 端推論需求透過網路 http post 向 AI Portal Server 提出。
2. AI Portal Server 看哪一個 AI 推論裝置有空，將推論需求透過網路 http post 向 AI 推論裝置提出。
3. AI 推論裝置收到推論需求後，如果深度學習模型不同，透過網路 http post 向 AI Portal Server 索取深度學習模型。
4. AI 推論裝置推論完成後，將結果以 JSON 格式回傳給 AI Portal Server 。
5. AI Portal Server 將結果以 JSON 格式回傳給 Client 端。

晶圓、封測、面板、銅箔、 PCB 、設備、安控等。
